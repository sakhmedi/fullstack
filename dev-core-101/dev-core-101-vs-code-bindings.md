## 1. **Ctrl + P**
Открывает быстрый поиск файлов в проекте.

## 2. **Ctrl + /**
Комментирует или раскомментирует выбранную строку.

## 3. **Ctrl + Shift + P**
Открывает палитру команд, где можно выполнить любую команду VS Code.

## 4. **Ctrl + B**
Скрывает или показывает боковую панель (Explorer).

## 5. **Ctrl + Shift + N / Ctrl + Shift + W**
Открыть новое окно VS Code / закрыть текущее окно.
