# Compiled and Interpreted Programming Languages

This file lists 8 programming languages and indicates whether they are compiled or interpreted.

| Language    | Type         | Notes                                                                 |
|------------|-------------|----------------------------------------------------------------------|
| C          | Compiled     | Translated into machine code by a compiler before execution. Fast performance. |
| Python     | Interpreted  | Executed line by line by an interpreter. Easy to test and debug.     |
| JavaScript | Interpreted  | Runs directly in the browser or Node.js. Used for interactive websites. |
| Java       | Both         | Compiled into bytecode, then interpreted by the Java Virtual Machine (JVM). |
| Go         | Compiled     | Compiled language, designed for scalable server applications.        |
| Ruby       | Interpreted  | Interpreted language, commonly used in web development (e.g., Rails). |
| C++        | Compiled     | Powerful compiled language for high-performance applications.        |
| PHP        | Interpreted  | Mainly used for server-side web development; executed by an interpreter. |
